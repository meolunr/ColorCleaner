# ColorCleaner
自用 HyperOS 官改制作工具

适用于 HyperOS Android 16 高通骁龙 8 Gen2 / 8 Gen3 平台机型

## 特性
- 禁用强制 Data 分区加密
- 禁用 AVB 验证引导
- 自定义打包内核镜像
- 自定义安装 KernelSU LKM
- 精简无用系统内置应用
- 空白 cust 分区
- 替换空白 Analytics 应用
- 禁用 V3 签名完整性验证
- 禁用关联启动对话框
- 禁用主题自动恢复
- 禁用历史通知
- 去除应用包管理组件广告
- 允许应用包管理组件安装系统应用
- 去除主题商店广告
- 免费使用主题
- 禁用设备名称敏感词检查
- 去除应用信息举报按钮
- 显示电池健康度
- 显示电池温度
- 禁用手机管家立即修复
- 显示详细耗电数据
- 禁用耗电项优化建议
- 禁用 Root 权限检查
- 去除危险操作倒计时确认
- 去除安全组件
- 显示网络类型设置
- 去除短信广告
- 禁用 Mrm 风险检测
- 忽略修改过的系统应用更新
- 更新系统应用为 ROM 打包时最新版本

## 系统更新
- 本工具支持在构建时更新系统应用为最新版本，避免用户在开机后需要进入应用商店二次更新，同时也可以避免在 super 分区存储一份无用的老版本 apk 文件，有限地起到节省空间的作用

  如需使用此功能，需通过 ADB 连接到设备，本工具将以设备上的系统应用为基准，更新 ROM 中的系统应用
- 本工具支持单独构建系统更新 Magisk / KernelSU 模块，用户可通过安装模块更新修改过的系统应用，在保留修改效果的同时体验官方的新特性

  模块仅更新需要被修改的系统应用，无需修改的系统应用不会被打包进模块内，用户可自行进入应用商店内更新
